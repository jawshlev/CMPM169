Experiment 7: Data Visualization & Networks

Description

With this experiment I sought to use the bart train api to emulate the little displays you see in bart stations that tell you the times of incoming trains.

Technical

BART API
Using the Bart API, I made a call to grab train information with the train station input, then I used an array with abbreviations which is how the API understands which station I am requesting information for, I found someone on YouTube who made a discord bot with the BART API so I used some of their code as a boilerplate and from there I simply added an input field, and buttons to toggle a map

Scrolling Text
I found an existing example of a p5 demo with scrolling text, I aimed to implement this functionality to bring the idea of a station to life

Reflection

This final product is far from what I really set out to do, I wanted to use an iFrame to place this page onto a 3d bart Kiosk, I wasn't able to figure it out but that's definitely something I want to figure out how to do later on for a personal project.
